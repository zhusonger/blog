# 安装rvm
1. 下载并执行命令，这里需要给终端设置代理, 参照[00:终端Terminal便捷操作](</terminal>)内的终端代理部分
	
	```shell
	curl -sSL https://get.rvm.io | bash -s stable
	```
	
2. 查看可安装的版本，在最上面

	```shell
	rvm list known
	```
	
3. 安装可安装的版本,如果出现错误,可以先尝试降低版本, 可能是新版本跟旧系统不兼容

	```shell
	rvm install 2.6.0
	```

4. 设置默认版本  

	```shell
	rvm use 2.6.0 --default
	```
